# atom-cc
## Array

Create array:
```
array = new int[len_of_array]
```
Modify array:
```
array[index]=expr
```
Access array:
```
access=array[index]
```
Length of array:
```
len_array=len(array)
```
Print array:
/!\ This function forbid the use of var named $array_nameshowarr
```
showarr(array)
```
